# Finance Platform Website

A modern, responsive website for a financial platform with a Home page and four product pages.

## Pages

- **Home** (`index.html`) – Overview and links to all products
- **Business Banking** (`business-banking.html`) – FDIC-insured business checking
- **Corporate Cards** (`corporate-cards.html`) – Smart corporate cards with spend controls
- **Global Banking** (`global-banking.html`) – International payments and multi-currency
- **Expense Management** (`expense-management.html`) – Automated expense reporting and reconciliation

## Running locally

Open `index.html` in a browser, or use a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .
```

Then visit `http://localhost:8000`.
